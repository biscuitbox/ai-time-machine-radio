# AI 타임머신 라디오 PRD

> 작성일: 2026-05-09
> 작성자: 본인
> 버전: 0.1

---

## 1. 개요 (One-liner)

> 책상 위에 두고 다이얼을 돌려서 1970~2100년 임의 시대의 한국어/영어 라디오 방송(AI 생성 DJ 멘트 + 시대 음악 + 빈티지 효과음)을 재생하는, 인테리어 겸 장난감 빈티지 라디오.

---

## 2. 사용자 / 시나리오

- **사용자**: 본인 + 가족·손님 (비기술자 포함, 어린이 사용 가능성 있음 → 케이스 안전 마감 필요)
- **사용 시나리오**: 작업 중 BGM으로 책상 위에 상시 ON. 손님 방문 시 다이얼 돌려가며 시대별 방송 시연. 하루 6~10시간 켜두는 패턴.
- **조작 방식**: 진짜 토글 전원 스위치, 연도 다이얼(로터리 인코더), 추가 노브 3개(볼륨 / 채널 / 한·영 언어 토글). 앱·음성 조작 없음.

## 부록 C. v0.1 GPIO·I2C 핀 할당 (잠정)

전제: Raspberry Pi 5, BCM 핀 번호 기준. KY-040은 모듈 내장 풀업 사용. SH1106은 I2C 주소 0x3C 또는 0x3D — 첫 부팅 시 `i2cdetect -y 1` 로 확인.

| 용도 | 핀(BCM) | 비고 |
|---|---|---|
| OLED SDA (I2C0) | GPIO2 | SH1106 |
| OLED SCL (I2C0) | GPIO3 | SH1106 |
| I2S BCLK (예약, v0.5+) | GPIO18 | v0.1에서 사용 금지 |
| I2S LRCLK (예약, v0.5+) | GPIO19 | v0.1에서 사용 금지 |
| I2S DIN (예약, v0.5+) | GPIO20 | v0.1에서 사용 금지 |
| I2S DOUT (예약, v0.5+) | GPIO21 | v0.1에서 사용 금지 |
| 연도 다이얼 CLK | GPIO17 | KY-040 |
| 연도 다이얼 DT | GPIO27 | KY-040 |
| 연도 다이얼 SW | GPIO22 | KY-040 압입 스위치 |
| 볼륨 CLK | GPIO5 | KY-040 |
| 볼륨 DT | GPIO6 | KY-040 |
| 볼륨 SW | GPIO13 | KY-040 (미사용 가능) |
| 채널 CLK | GPIO16 | KY-040 |
| 채널 DT | GPIO26 | KY-040 |
| 채널 SW | GPIO12 | KY-040 (미사용 가능) |
| 언어 CLK | GPIO23 | KY-040 |
| 언어 DT | GPIO24 | KY-040 |
| 언어 SW | GPIO25 | KY-040 — 한·영 토글 트리거로 활용 가능 |
| 전원 토글 입력 | GPIO4 | 토글 SW 상태 감지 → systemd 셧다운 트리거 |
| 전원 라치 keep-alive | GPIO7 | Pi가 부팅 후 HIGH로 유지 → 셧다운 마지막 단계에 LOW로 떨어뜨려 라치/로드 스위치 차단 |
| 예비 | GPIO8~11(SPI), GPIO14~15(UART) | 향후 LED·확장용 |

비고:
- `dtparam=i2c_arm=on`, `dtparam=i2s=on` 활성화 (`/boot/firmware/config.txt`).
- I2C 주소 충돌 가능성: SH1106 OLED 외에 ADS1115·MCP23017 등 다른 I2C 장치를 추가할 경우 0x3C/0x3D와 충돌하지 않는지 재확인.
- 4개 KY-040 인코더가 GPIO 4×3=12핀을 차지. SW는 미사용 시 비워둘 수 있음.
