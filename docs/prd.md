# AI 타임머신 라디오 PRD

> 작성일: 2026-05-09
> 작성자: 본인
> 버전: 0.1

---

## 1. 개요 (One-liner)

> 책상 위에 두고 다이얼을 돌려서 1970~2100년 임의 시대의 한국어/영어 라디오 방송(AI 생성 DJ 멘트 + 시대 음악 + 빈티지 효과음)을 재생하는, 인테리어 겸 장난감 빈티지 라디오.

---

## 2. 사용자 / 시나리오

- **사용자**: 본인 + 가족·손님 (비기술자 포함, 어린이 사용 가능성 있음 → 케이스 안전 마감 필요)
- **사용 시나리오**: 작업 중 BGM으로 책상 위에 상시 ON. 손님 방문 시 다이얼 돌려가며 시대별 방송 시연. 하루 6~10시간 켜두는 패턴.
- **조작 방식**: 진짜 토글 전원 스위치, 연도 다이얼(로터리 인코더), 추가 노브 3개(볼륨 / 채널 / 한·영 언어 토글). 앱·음성 조작 없음.

## 3. 기능 요구사항

### MUST (필수)

**v0.1 단계 필수**
- [ ] 다이얼로 1970~2100 연도 선택, 디스플레이(OLED)에 현재 연도 표시
- [ ] 진짜 토글 스위치로 전원 ON/OFF
- [ ] 다이얼·노브 입력이 시리얼 로그에 정확히 출력 (`year=1985 ch=news lang=ko vol=70`)

**v0.5 단계 필수**
- [ ] 로컬 LLM이 해당 연도의 한국어 또는 영어 DJ 멘트·뉴스 톤 텍스트 생성
- [ ] 로컬 TTS(Piper)로 멘트 음성 합성·재생
- [ ] 라디오 노이즈/주파수 효과음 (다이얼 회전 중 지지직, 정착 시 페이드)
- [ ] 부팅 시 진공관 워밍업 효과음
- [ ] 멘트 시작·종료 자동 삽입 (`"이상 1972년 KBS 9시 뉴스였습니다"` 등)
- [ ] 사전 캐싱: 1970~2026 × 한/영 × 채널별 멘트를 백그라운드에서 미리 생성·저장
- [ ] 채널 노브로 채널 간 전환
- [ ] 언어 토글 노브로 한·영 즉시 전환

**v1.0 단계 필수**
- [ ] 그 시대 음악 재생 (로컬 음원 라이브러리)

- [ ] USB-C 5V 상시급전 + 18650 1셀 백업 배터리 (이동·정전 시 1~2h)
- [ ] 며칠 연속 무인 동작 (systemd `Restart=always` + 메모리 모니터링)
- [ ] 본 3D 프린트 케이스 완성 (Bambu P1S 출력, 환기구 포함)

- [ ] 아날로그 바늘 시각효과
- [ ] 외부 인터넷에서 라디오로 원격 접근

## 4. 물리적·환경적 제약

| 항목 | 내용 |
|---|---|
| 폼팩터·크기 | 약 25(W) × 15(H) × 12(D) cm 내외, 부품 확정 후 조정 |
| 설치 위치 | 책상 위 |
| 환경 | 실내 상온, 직사광선·먼지 노출 적음 |
| 전원 | USB-C 5V 상시급전 (PD 어댑터 27W 이상 권장: Pi 5 + 액티브 쿨러 + 앰프·스피커 합산). 백업: 18650 1셀 + 충전 IC(TP4056) + 5V 부스트 |
| 배터리 운용 시 교체 주기 | 백업용 18650 1셀, 2~3년 후 교체 |
| 소음/발열 허용 | Pi 5 액티브 쿨러 팬 소음 허용 (책상 위 BGM 환경). 표면 발열은 만질 수 있는 수준(~40°C 이하) 유지 |
| 외관·케이스 | Bambu Lab P1S로 3D 프린트. 빈티지 라디오 외형(우드 텍스처 도색 또는 컬러 프린트). 환기구·40mm 팬 마운트 포함. 어린이 접촉 시 안전한 마감(날카로운 모서리 제거, 배터리 셀 노출 방지) |

## 7. 단계 목표

### v0.1 (MVP)
- **Done 정의**: 다이얼 회전 시 OLED에 1970~2100 연도가 정확히 표시되고, 채널·언어·볼륨 노브 입력이 시리얼 로그에 `[t=12.3] year=1985 ch=news lang=ko vol=70` 형식으로 출력. 토글 스위치 ON/OFF 작동. 임시 케이스(종이 상자 또는 MDF). 음원·AI는 미구현.
- 검증 단위: (a) 전원만, (b) OLED I2C blink, (c) 인코더 단독 카운트, (d) 노브 3개 단독, (e) 토글 스위치, (f) 통합.

### v0.5
- **Done 정의**: 다이얼을 변경하면 10초 내 한국어 또는 영어 AI DJ 멘트 + 라디오 노이즈 효과가 재생됨. 전원 켤 때 진공관 워밍업 효과음, 멘트 끝에 시작/종료 멘트 자동 삽입. 1970~2026 × 한/영 × 채널 1개의 사전 캐싱이 백그라운드에서 점진 완성됨. 임시 3D 프린트 케이스(외형은 단순). 음악은 미구현.

### v1.0
- **Done 정의**: 시대 음악 재생(로컬 라이브러리), 채널 노브로 뉴스/팝/광고/클래식 전환. 본 3D 프린트 케이스(빈티지 외형) 완성. USB-C 상시급전 + 18650 1셀 백업으로 1~2h 무전원 작동. 7일 연속 무인 동작 검증(프로세스 죽지 않음, 메모리 점유 안정).

### v2.0 (선택, 향후)
- 완전 오프라인 모드 (모델·음원 모두 SSD/SD에서, Wi-Fi 없이 동작)
- 미래 모드(2027~2100) 정교화: 시대별로 AI가 가상 사회·기술·사건의 일관성 유지하는 메타-월드 빌딩
- 청취 패턴 학습: 자주 듣는 연도·채널 프로파일링하여 캐시 우선순위 자동 조정
